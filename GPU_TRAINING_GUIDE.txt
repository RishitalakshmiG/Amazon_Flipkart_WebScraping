════════════════════════════════════════════════════════════════════════════════
  TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

If GPU is not detected:
  1. Check NVIDIA GPU installed:
     $ nvidia-smi
  2. If command not found, download CUDA Toolkit:
     https://developer.nvidia.com/cuda-downloads
  3. Reinstall PyTorch CUDA:
     $ pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

If download is slow:
  • PyTorch is 2.8 GB - may take 10-30 minutes
  • Can check progress in terminal
  • Once installed, training will be VERY fast (2-4 min)

If training fails:
  1. Verify GPU: $ python gpu_setup.py
  2. Check memory: $ nvidia-smi
  3. Run on CPU: train_gpu.py auto-fallback if GPU fails
